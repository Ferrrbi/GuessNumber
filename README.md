# GuessNumber - Zgadnij liczbę
Odnajdź w sobie wewnętrznego hazardziste!

## Uruchomienie
Gre włączamy przez kompilator C++, na przykład ***Visual Studio***.

## Sterowanie
Po uruchomieniu gry ukaże się nam *interface graficzny* z którego możemy wybrać trzy opcje, gra, tabela wyników lub wyłączenie gry.
W samą gre gramy przy użyciu klawiszy numerycznych, wpisujemy liczbę i zatwierdzamy ją Enterem.

## Możliwości 
Gra posiada opcje wyboru poziomu trudności gry, łatwy, średni oraz trudny. Po wygraniu naszej rozgrywki wynik zostanie zapisany w tabeli, którą będziemy mogli później podejrzeć.
